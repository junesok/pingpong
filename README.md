# Pingpong Rally

Keep the ball in the air with your paddle. Track your hits and survival time.

## Features

- 60fps Canvas rendering with physics-based ball movement
- Mouse and touch controls
- Hit effects — particle burst, screen shake, paddle glow, ball trail
- Slow motion as the ball nears the bottom
- Game over screen with current score, personal best, and share button
- Dark and light mode with system preference detection

## Built with

- [Next.js](https://nextjs.org) 16 (App Router)
- TypeScript
- HTML5 Canvas API
- Tailwind CSS
- Google Fonts — Playfair Display, Inter

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
